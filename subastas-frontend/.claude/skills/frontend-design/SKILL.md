# Skill: Frontend Design con Tailwind CSS

## Propósito
Guiar al modelo en la creación de interfaces de usuario modernas, estéticas, limpias y profesionales utilizando React y Tailwind CSS, asegurando la consistencia visual y la responsividad.

## Reglas de Diseño
- **Estructura limpia:** Usar componentes semánticos de HTML5 y layouts basados en Flexbox o CSS Grid.
- **Paleta de colores:** Mantener un contraste alto. Evitar saturar con colores estridentes. Para subastas, usar un esquema limpio (fondos blancos/grises claros, tarjetas con bordes suaves, acentos en azul o esmeralda para botones de acción).
- **Interactividad:** Los estados `:hover`, `:focus` y `:active` deben estar presentes en todos los botones y elementos interactivos.
- **Responsividad:** Diseñar pensando primero en móviles (`mobile-first`) y escalar usando los breakpoints de Tailwind (`md:`, `lg:`).
- **Tipografía:** Tamaños de texto claros y legibles usando clases como `text-sm`, `text-base`, `text-lg` y `font-semibold` para títulos.

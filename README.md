## Proyecto
Sistema de Subastas.

## Objetivo
El objetivo de este proyecto es implementar una plataforma que permita administrar subastas de manera eficiente, brindando funcionalidades para la publicación de productos, gestión de usuarios, realización de pujas y determinación automática de los ganadores al finalizar cada subasta.
